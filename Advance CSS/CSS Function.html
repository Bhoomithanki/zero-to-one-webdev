Whats the use of var() Function in CSS?

The var() function is used to the value of a CSS variable.
A good way to use CSS variables is when it comes to the colors of your design. Instead of copy and paste the same colors over and over again, you can place them in variables.

First of all: CSS variables can have a global or local scope.

  • Global variables can be accessed/used through the entire document.
  • while local variables can be used only inside the selector where it is declared.

    • To create a variable with global scope, declare it inside the :root selector. The :root selector matches the document's root element.
    To create a variable with local scope, declare it inside the selector that is going to use it.
    The following example is where we use the var() function.

    In this case Instead of using the color again and again we give the color to the local varaible in the parent
    FOR PRIMARY BUTTON :--primary--bg--color: #00b7ff;
    FOR SECONDARY BUTTON :--secondary--bg--color: #6c757d;
