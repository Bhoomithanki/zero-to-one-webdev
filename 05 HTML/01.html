What's SEO ?

SEO stands for Search Engine Optimization.
Engine means Interpreater or compiler while optimization means the process of modifying a system to make some features of it work more efficiently .
It's the process of improving your website to increase its visibility in Google, Microsoft Bing, and other search engines .
In HTML comman way to help Search Engine is to use the following :

  • Use Title Tag

    • One of the most important HTML tags for SEO is that of the title. This is the label of your content and how searches see your page in Google and Bing. Every result you see in a search engine is derived from the title tag.

  • Meta Tags

    • Another vital HTML tag for SEO is the meta description. Like titles, this information shows in search results of Google. Take a look under the link for any search. It's best if you use your keyword in the title as well as the meta description tag. It improves how search engines list your content.

  • Header Tags

    • Header tags are often used to break up the content to make it easier to read.Headers make it easier for these people to find what they’re looking for quickly.The h1 to h6 behave as following for SEO :

    • h1 - usually reserved for webpage titles.
    • h2 - highlights the topic of the title.
    • h3 - reflects points in regards to the topic.
    • h4 -supports points from h3.
    • h5 – not often used, but great for supporting points of h4.

  • Alt Tags

    • Unfortunately, search engines cannot determine what images are trying to convey.
    That’s when the ALT tag comes into play.This tag tells web browsers what image to show in the content. the alt attribute of this tag tells search engines the picture is about.And this could reduce exposure during image searches.

  • Language Meta Tags

    • language meta stage which tells search engines the language of the page.
