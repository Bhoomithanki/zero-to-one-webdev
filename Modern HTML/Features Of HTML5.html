Features of HTML5

What's HTML5?

HTML5 is the latest version of HTML developed by W3C (World Wide Web Consortium). Goal was to create a new verision of HTML that would meet the needs of modern applications.

Let's talk about its Features:

  • Improved Multimedia support

    • Allow Integrating multimedia element such as <audio> and <video> directly into webpage without the need of plugins like flash

  • Canvas element

    • It is a graphics library for drawing and manipulating images and video in the browser.

  • Geolocation API

    • Is an API for obtaning the user's location , enabling web appplication to offer location-based services.

  • Local Storage

    • Allows of storage of data of the user's device,improving performance and reducing the need for round-trips to the server.

  • New Structure element

    • The new structure elements are: header, footer, nav, section, aside makes easier to structure and organize content on a web page.

  • Form Improvements

    • <datalist> - A list box containing pre-defined options for text input controls(newly added input types are date ,time and color).have attributes such as required and autofocus.

  • Accessibility Improvements

    • They provide features to make web pages more accessible for people with disabilities.
