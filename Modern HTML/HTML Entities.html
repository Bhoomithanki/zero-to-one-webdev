Html Entities

HTML Entities are reserved characters used in HTML Doc.(Generally not seen in typical keyboard)
Purpose of HTML Entities are :

  • Displaying Special characters.(Character that have special meaning in HTML or characters which cannot be typed easily)
  • Handling Reserved characters.(Uses special character for syntax such as < and > which are not enable to render on web pages while preserving the structure of the web page.)

Generally entity code is enclosed in ampersand and semicolons to differentiate it from regular text.
They are many HTML Entities some of them are as follow:
  • less than

    • Example: <

  • Greater than

    • Example: >

  • ampersand

    • Example: &

  • double quatation mark

    • Example:"

  • single quatation mark

    • Example:'

  • copyright

    • Example:©

  • registered trademark

    • Example:®
