The z-index property specifies the stack order of an element. An element with greater stack order is always in front of an element with a lower stack order.

Values of z-index are as follow:

  • VALUE AUTO : The box does not establish a new local stacking context. The stack level of the generated box in the current stacking context is 0.
  • VALUE NUMBER: Sets the stack order of the element. Negative numbers are allowed.
  • VALUE INITIAL : Sets this property to its default value.
  • VALUE INHERIT : Inherits this property from its parent element.
