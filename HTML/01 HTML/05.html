Doctype declaration in html

Doctype declartion in html means 'document type declaration' they are used to declare which version of a html we are using.
They are different version of html has different doctype decleration.
It can be written as < ! DOCTYPE html > .
They are generally written on the begning of html document to specifices which version of a html we are using they are used by browser for better understanding of the version which we have use so it helps for better rendering of a content on the web pages.
