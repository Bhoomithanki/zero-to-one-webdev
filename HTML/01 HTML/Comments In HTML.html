Comments in HTML

In simple words Comments are generally used when we don't want some specific content to render on the web page either it just removes the rendering of that specific content into the webpage.

The comment in html is done by "angular brackets opens ! -- content -- angular bracket close Tag " and in windows we can also use shortcut ' Ctrl + / '.

The commenting content is there only but it just doesn't render on the web page you can check the comment part by just right click on the webpage and click on inspect
